# Data Analytics Internship Repository

This repository contains **official learning materials, assignment guidelines, and quiz references** from a **Data Analytics Internship bonus chapter** under the broader UX & Game Design Internship Program.

It is a **supplementary learning module**, not a standalone internship, added to strengthen foundational knowledge in data analysis alongside UX and Game Design.

---

## Assignments

> The `assignments/` folder includes official internship assignment guidelines provided during the program.
>
> Typical files include:
>
> A01 Project Guidelines.pdf  
> A02 Project Guidelines.pdf  
> A03 Project Guidelines.pdf  
> …and so on  
>
> These documents contain instructions, task descriptions, and requirements but do not include completed submissions or full project documentation.
